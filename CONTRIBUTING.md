# Contributing

This repository is mainly an experimental workflow log and ROS 2 integration demo.

Contributions are welcome in these areas:

1. Isaac Sim scene setup notes;
2. ROS 2 Humble compatibility fixes;
3. FAST-LIO configuration improvements;
4. Nav2 integration based on 3D LiDAR / IMU localization;
5. Troubleshooting logs and reproducible test cases.

Please avoid uploading large binary files such as rosbag, point cloud maps, or Isaac Sim USD scenes directly to the repository.
